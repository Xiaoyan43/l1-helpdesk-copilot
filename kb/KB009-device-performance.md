# KB009 — Slow laptop / device performance

**Applies to:** Windows laptops, general slowness, high CPU/disk usage.

## Symptoms
- Laptop takes minutes to open apps or respond.
- Fan runs constantly; everything feels sluggish.
- Performance degraded over days/weeks (not a single app crash).

## L1 resolution steps
1. Ask when slowness started and whether restarts help temporarily.
2. Check free disk space (target ≥ 15% free on system drive); clear temp files if low.
3. Review startup apps and disable non-essential items; confirm Windows Update is not stuck mid-install.
4. Run a quick malware scan if the user reports pop-ups or unknown processes.
5. If still slow after reboot + cleanup, schedule a deeper health check or reimage per endpoint policy.

## When to escalate
- Suspected malware or encryption alert → escalate to Security (see **KB006**).
- Hardware failure signs (clicking drive, overheating shutdown) → escalate to Endpoint / L2 hardware.
